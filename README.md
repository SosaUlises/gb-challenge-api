# Grupo Blanco Challenge - Backend

API REST desarrollada en .NET 8 para el simulador empresarial Grupo Blanco Challenge.

El proyecto modela situaciones reales de gestión inspiradas en la operatoria de Grupo Blanco, empresa argentina dedicada a la importación y distribución de marcas internacionales de calzado.

## Funcionalidades

* Inicio de partidas
* Gestión de sesiones de juego
* Sistema de decisiones estratégicas
* Cálculo de indicadores empresariales
* Evaluación final de desempeño
* Ranking global de jugadores
* Persistencia en PostgreSQL (Neon)

## Indicadores de Gestión

Cada decisión impacta sobre:

* Rentabilidad
* Clientes
* Clima Organizacional
* Imagen de Marca
* Eficiencia Operativa
